# 🚀 4 Farklı AI ile Web Tasarımı
Bu proje, modern yapay zeka araçlarının web tasarım yeteneklerini ölçmek ve karşılaştırmak amacıyla oluşturulmuş bir üniversite çalışmasıdır. Aynı tasarım hedefleri doğrultusunda Gemini, Claude, Antigravity ve Cursor modelleri kullanılarak 4 farklı web sayfası geliştirilmiştir.

---

## 📋 Proje Hakkında
Bu çalışmada, her bir AI modeline aynı direktifler verilerek oluşturulan HTML ve CSS yapıları incelenmiştir. Projenin temel amacı:
AI modellerinin kod yazma verimliliğini karşılaştırmak.
Tasarım estetiği ve kullanıcı deneyimi (UX) farklarını gözlemlemek.
Modern geliştirme süreçlerinde AI kullanımının avantajlarını dökümante etmek.

---
